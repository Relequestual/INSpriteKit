## Requirements

iOS 7+, ARC enabled

Needs the following Frameworks:
- SpriteKit
- GLKit


## Installation

Add the following line to your Podfile:

	pod 'INSpriteKit', :git => 'https://github.com/indieSoftware/INSpriteKit.git'
